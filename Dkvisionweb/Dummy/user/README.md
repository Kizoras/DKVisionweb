# Situs Demo Jurusan DKV [Made by Project Team 8]

Demo situs statis untuk jurusan Desain Komunikasi Visual.

Fitur:
- Halaman: Home, About, News, Curriculum, Portfolio, Products, Contact (template tersedia)
- Login admin (client-side demo). Login: admin / admin123
- Setelah login, admin dapat menambah, mengedit, dan menghapus berita, portofolio, dan produk.

UI/UX Upgrade:
- Tampilan telah diperbarui menjadi card-based modern (header gradien, kartu untuk berita/portofolio/produk, tombol yang lebih jelas).

Cara menjalankan:
1. Buka `index.html` di browser (double-click atau buka via Live Server di VS Code).
2. Klik Login lalu masukkan kredensial demo.

Catatan: Ini adalah demo berbasis front-end. Untuk produksi, ganti mekanisme auth dan simpan data pada server.

Paste image / URL:
- Saat sudah login sebagai admin, kamu bisa menempel (Ctrl+V) gambar langsung dari clipboard (mis. screenshot) atau menempelkan URL gambar (copy image address) di mana saja pada halaman. Sistem akan otomatis membuka modal tambah portofolio dan menampilkan preview gambar.
